## 1. 한 줄 소개

**120BPM 박자에 맞춰 왼손으로 리듬 패턴을 입력해 골드를 벌고·적을 쓰러뜨리며, 그 사이 오른손으로 타워를 깔아 기지를 지키는 2D 탑뷰 디펜스. 2분 생존이 목표.**

---

## 2. 핵심 컨셉

| 요소 | 내용 |
|------|------|
| 장르 | 2D 탑뷰 디펜스 + 원버튼 리듬 |
| 플랫폼 | PC (키보드 + 마우스) |
| 차별점 | 양손 협응 — 왼손 리듬, 오른손 타워 배치 **동시 가능** |
| 리듬 기준 | **2/4 마디** · 기본 **1초/마디** (120 BPM) · `measureDurationSeconds` 조절 · 가속 **×0.8** |
| 적·스폰 | 가속 중에도 **적 이동·스폰 속도 불변** |
| 타워 사격 | **평타 타워만** BeatClock 박자 동기 (가속 150BPM) |
| 전투 | 웨이브 없음, 지속 스폰 |
| 안전장치 | **평타 타워**만 박자 DPS · **1.2s** 무입력 시 Fallback 30% |
| **승리** | **2분(120초) 생존** (Core HP > 0) |
| **패배** | Core HP = 0 |

## 3. 양손 조작

> 상세: [CONTROLS.md](./CONTROLS.md)

### 왼손 — Spacebar

- **4종 리듬 패턴** → `PERFECT` / `GOOD` / `MISS`
- **골드 채굴·평타** — 쿨타임 없음
- **강공격·가속** — 쿨타임 있음 (§4.5)

### 오른손 — 마우스

- **HUD에서 타워 종류 선택** (평타 / 강공격 / 버프) → **좌클릭:** 슬롯 설치
- **타워 클릭** → 근처 **판매** 버튼 (환급 50%)
- **점유 슬롯에 새 타워 설치** → 기존 타워 **자동 판매(50%)** 후 교체

### 동시 입력 (확정)

> **Space 리듬 입력 중에도 마우스 클릭으로 타워 배치 가능.** Pause(ESC) 중에는 불가.

### 시스템

| 입력 | 동작 |
|------|------|
| **ESC** | 일시정지 — 계속 / **재시작** / 박자 연습 / 설정 / 시작 화면 |
| **R** | **미할당** — 재시작은 Pause 메뉴 버튼 |

## 4. 리듬 패턴 4종 (2/4 마디 · Reference 1초 기준)

> **마디:** **2박(2/4)** · 기본 **마디 길이 1초** (1박=0.5s) · Phase A 테스트 씬은 **4초**  
> **조절:** `measureDurationSeconds` — **2초**로 올리면 아래 시각·박자·판정창 **전부 ×2**  
> **판정:** 각 입력 시각 **±0.22s × (마디길이/1초)** (PERFECT ±0.11s)  
> **가속(BPMBoost):** `EffectiveMeasureDuration` **×0.8** (BeatTower·메트로놈·판정 동기)  
> **흐름 (구현 확정):** 한 마디(2박) 동안 입력 **수집** → **마디 종료(사이클 끝)** 시 일괄 채점·효과 적용 → 다음 마디 **초기화** · 2/2박(중간)에는 채점 없음  
> **조기 1박:** 사이클 **끝 ±PERFECT** · **현재 사이클 입력 0개일 때만** 이월 (Gold 2타 등은 후반 입력이 빠지지 않음)

### 🟡 패턴 A — 골드 채굴 (GoldPulse)

| 항목 | 내용 |
|------|------|
| 입력 | **쿵! 쿵!** (정박 2연) |
| 시각 (1초 마디) | **0 → 0.5** (2타) |
| 효과 | **골드 +10** (PERFECT/GOOD, MISS 시 0) |
| **쿨타임** | **없음** |

> 경제·타워 가격 등: [BALANCE.md](./BALANCE.md)

### 4.7 입력 시각화 (RhythmTimelineUI · Phase A)

| 항목 | 내용 |

## 5. 디펜스 · 타워 (3종)

> **맵:** [MAP.md](./MAP.md) · **수치:** [BALANCE.md](./BALANCE.md)

### 5.1 공통

| 항목 | 내용 |
|------|------|
| 본진 | `(0, -3.5)` · HP **25** |
| 스폰 | S1 / S2 → 합류 초크 `(0,-1)` |
| **최대** | **8기** (종류 합산) · 시작 **20G** |
| 사거리 | **원형 2.5** · 경로 **선두** 1마리 (평타 타워만) |
| 판매 | 클릭 → 판매 버튼 · **50% 환급** |
| 교체 | 점유 칸에 설치 → 기존 **자동 판매(50%)** |

### 5.2 타워 종류

| 종류 | 비용 | 역할 |
|------|------|------|
| **평타 타워** (Beat) | **20G** | **박자마다 2 dmg** (최근 **1.2s** Space 입력) · 무입력 Fallback **0.6** · **RhythmShot** 즉시 1발 |
| **강공격 타워** (Strike) | **30G** | 자동 사격 **없음** · **OverloadStrike** 성공 시에만 범위 **8 dmg** (적당 1회) |
| **버프 타워** (Boost) | **25G** | 자동 사격 **없음** · **BPMBoost** 성공 시에만 범위 **4 dmg** (적당 1회) + 플레이어 가속은 별도 |

> 강공격·버프 타워는 **해당 커맨드 입력·성공 시에만** 발동. 평타 타워만 상시(박자) DPS.

### 5.3 박자 DPS (A안 — 평타 타워만)

| 상태 | 평타 타워 |
|------|-----------|
| 최근 **1.2s** 내 Space 입력 | 박자마다 **2 dmg** |
| **1.2s** 완전 무입력 | 박자마다 **0.6 dmg** (Fallback) |
| 가속(150BPM) 중 | 박자 간격 **0.4s** (동일 규칙) |

### 5.4 적 · 연출

| 항목 | 내용 |
|------|------|
| 몹 | **8분음표** / **강박** (디자인 TODO) |
| **박자 연출** | 이동 중 **120BPM**에 맞춰 스프라이트 **살짝 커졌다 작아짐** (탄성) — 전투 수치 무관 |

### 5.5 스폰 스케줄 (2분)

> **연속 ramp** — [BALANCE.md §7.1](./BALANCE.md) · 구간 급변 없이 smooth step

| t (대략) | S1/S2 | 비고 |
|----------|-------|------|

## 7. 씬 · 흐름 · 튜토리얼

> 상세: [FLOW.md](./FLOW.md)

### Pause (ESC)

- **적·타워·스폰·Core** 정지 (`timeScale = 0`)
- **BeatClock·메트로놈·Rail** — **계속 재생**, Rhythm 볼륨 **40%**
- 계속 · 재시작 · **박자 연습** · 설정 · 시작 화면

### Practice (PracticeScene)

- **별도 씬** — Rail + Scroll 4패턴 · ms 오차 · 쿨 비활성
- **StartScene** · **Pause**에서 진입 — Pause 진입 시 **Additive Load** → 종료 시 Pause 복귀 ([FLOW.md §6](./FLOW.md))

### 결과 화면

- Victory(**2:00**) / Game Over
- **총점 · 등급** — [BALANCE.md §10](./BALANCE.md)
- 재시작 · 시작 화면

**표시 예:** `CLEAR — 15,025 (A)` · HP / 처치 / PERFECT·GOOD·MISS
