## PC 조작표

| 입력 | 동작 |
|------|------|
| **Space 및 미할당 키** | 리듬 패턴 (4종) — ESC·Tab·Shift/Ctrl/Alt·마우스 제외 |
| **HUD 타워 선택** | 평타 / 강공격 / 버프 |
| **마우스 좌클릭** | 빈 슬롯 설치 · **점유 슬롯 교체** (기존 50% 판매) |
| **타워 클릭** | **판매** 버튼 표시 → 50% 환급 |
| **Tab** | Rhythm Scroll 확대/축소 |
| **ESC** | Pause |

---

## 양손 동시 입력

Space·**기능 미할당 키보드 키** 리듬 입력 중 **슬롯 클릭·타워 판매** 가능. Pause 중 불가.
