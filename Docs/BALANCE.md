## 3. 타워 3종

### 3.1 공통

| 항목 | 값 |
|------|-----|
| **사거리** | **2.5** 유닛 (원형) |
| **타겟** | **평타 타워만** — 사거리 내 경로 **선두** 1마리 |

### 3.2 평타 타워 (BeatTower)

| 항목 | 값 |
|------|-----|
| 비용 | **20G** |
| **박자 DPS (A안)** | Lv1 **2** / Lv2 **2.8** / Lv3 **3.6** dmg (최근 **1.2s** Space 입력) |
| **Fallback** | Lv1 **0.6** / Lv2 **0.9** / Lv3 **1.2** (1.2s 완전 무입력) |
| **RhythmShot** | 성공 시 **평타 타워만** 즉시 **ActiveDamage** 1발 |
| **150BPM** | 박자 **0.4s** (동일 규칙) |

> Strike/Boost 타워는 **박자 사격·Fallback 없음**.

### 3.3 강공격 타워 (StrikeTower)

| 항목 | 값 |
|------|-----|
| 비용 | **30G** |
| 자동 사격 | **없음** |
| **OverloadStrike** 성공 | 모든 Strike 타워 **원형 범위 합집합** 안 적에게 **8 dmg 1회** (적당 **중복 없음**) |

### 3.4 버프 타워 (BoostTower)

| 항목 | 값 |
|------|-----|
| 비용 | **25G** |
| 자동 사격 | **없음** |
| **BPMBoost** 성공 | Boost 타워 범위 합집합 안 적 **4 dmg 1회** (중복 없음) + 플레이어 **150BPM 6s** (기존과 동일) |

### 3.5 스킬 dmg 중복 규칙

| 스킬 | 규칙 |
|------|------|
| OverloadStrike | 이번 발동당 **적 1마리 = 최대 8 dmg 1회** (Strike 범위 여러 개 겹쳐도) |
| BPMBoost (Boost 타워) | 이번 발동당 **적 1마리 = 최대 4 dmg 1회** |
| 강공격+버프 동시 | **별도 이벤트** — 각각 dedupe |

| 웨이브 | HP | 속도 | Core | Gold | 기믹 |

## 7. Core · 스폰

| 항목 | 값 |
|------|-----|
| **Core HP** | **3** (8분음표 3마리 도달 시 패배) |
| 필드 max | **78** |
| 시작 | **3초** 배치 여유 (스폰 delay 3s, 선택) |
| 이동 | 경로 waypoint + **lane offset** (±0.65) — 군집 |

### 7.1 스폰 스케줄 (2분)

> **연속 ramp** — 구간별 급변 없이 `t = 경과/120s` 기준 smooth step 보간.

| t (대략) | 간격 | S1/S2 | 마리/측 | 8분:강박 |
|----------|------|-------|---------|---------|
| 0~30s | **1.9→1.5s** | 주로 번갈아 | 1 | ~5% |
| 30~70s | **1.5→1.0s** | 번갈아↔동시 혼합 | 1 | ~15→25% |
| 70~100s | **1.0→0.75s** | 동시 비율↑ | 1→2 | ~25→32% |
| 100~120s | **0.75→0.62s** | 동시 | 2 | ~35% |

- **엘리트:** **60s · 90s** — §4.1 (일반 스폰과 별도)
- **Simultaneous:** t 28%부터 확률 상승, t 82%쯤 거의 항상 양쪽 동시
- **Count/side:** t 68%부터 1→2마리 ramp (t 97%쯤 2 고정)
