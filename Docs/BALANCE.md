## 3. 타워 3종

### 3.1 공통

| 항목 | 값 |
|------|-----|
| **사거리** | **2.5** 유닛 (원형) |
| **타겟** | **평타 타워만** — 사거리 내 경로 **선두** 1마리 |

### 3.2 평타 타워 (BeatTower)

| 항목 | 값 |
|------|-----|
| 비용 | **20G** |
| **박자 DPS (A안)** | 최근 **1.2s** Space 입력 → 박자마다 **2 dmg** |
| **Fallback** | **1.2s** 완전 무입력 → 박자마다 **0.6 dmg** |
| **RhythmShot** | 성공 시 **평타 타워만** 즉시 **2 dmg** 1발 |
| **150BPM** | 박자 **0.4s** (동일 규칙) |

> Strike/Boost 타워는 **박자 사격·Fallback 없음**.

### 3.3 강공격 타워 (StrikeTower)

| 항목 | 값 |
|------|-----|
| 비용 | **30G** |
| 자동 사격 | **없음** |
| **OverloadStrike** 성공 | 모든 Strike 타워 **원형 범위 합집합** 안 적에게 **8 dmg 1회** (적당 **중복 없음**) |

### 3.4 버프 타워 (BoostTower)

| 항목 | 값 |
|------|-----|
| 비용 | **25G** |
| 자동 사격 | **없음** |
| **BPMBoost** 성공 | Boost 타워 범위 합집합 안 적 **4 dmg 1회** (중복 없음) + 플레이어 **150BPM 6s** (기존과 동일) |

### 3.5 스킬 dmg 중복 규칙

| 스킬 | 규칙 |
|------|------|
| OverloadStrike | 이번 발동당 **적 1마리 = 최대 8 dmg 1회** (Strike 범위 여러 개 겹쳐도) |
| BPMBoost (Boost 타워) | 이번 발동당 **적 1마리 = 최대 4 dmg 1회** |
| 강공격+버프 동시 | **별도 이벤트** — 각각 dedupe |

## 10. 점수 산정

> 요약 [PLAN.md §10](./PLAN.md#10-점수-요약) · UI [FLOW.md §8](./FLOW.md)

| 축 | 클리어 |
|----|--------|
| 생존 | +5,000 (**2분** 클리어) |
| 방어 | HP × 200 |
| 전투 | 8분×80 + 강박×200 |
| 리듬 | PERFECT×15 + GOOD×5 |

**패배:** 전투+리듬 + `floor(생존초 × 10)`

**등급:** S 18,000+ / A 14,000+ / B 10,000+ / C 클리어 / D 패배

### 10.1 예시 (2분 클리어)

HP 16 · 8분 28 · 강박 8 · PERFECT 38 · GOOD 22 → **~12,800 (B)**
